StreamDataset
==============

:py:class:`~squirrel.dataset.stream_dataset.StreamDataset` defines the interface for datasets that provide asynchronous
accessing of their items.

A :py:class:`~squirrel.dataset.stream_dataset.Fetcher` is used to read and write items from/to a dataset.
